
# Exercícios em C – Vetores e Matrizes

Este repositório contém três exercícios desenvolvidos em **Linguagem C** para praticar conceitos básicos de programação como vetores, matrizes e estruturas de repetição.

## 📂 Estrutura do Projeto

```
trabalhos-c-programacao
│
├── matriz_4x5.c
├── matriz_invertida_6x3.c
├── soma_vetores.c
└── README.md
```

## 🧠 Exercícios

### 1. Matriz 4x5
Programa que solicita ao usuário os valores de uma matriz **4x5** e exibe todos os elementos digitados.

### 2. Matriz 6x3 Invertida
Programa que gera automaticamente uma matriz **6x3** com valores de **0 a 99** e depois imprime a matriz em **ordem inversa**.

### 3. Soma de Vetores
Programa que recebe dois vetores com **5 posições** e gera um terceiro vetor contendo a soma dos elementos correspondentes.

## ▶️ Como Compilar

Usando **GCC**:

```bash
gcc nome_do_arquivo.c -o programa
```

Exemplo:

```bash
gcc soma_vetores.c -o soma
./soma
```

## 🎯 Objetivo

Praticar:
- Vetores
- Matrizes
- Estruturas de repetição (`for`)
- Entrada e saída de dados (`scanf` / `printf`)
