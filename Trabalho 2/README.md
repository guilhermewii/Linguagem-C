
# Exercícios em Linguagem C

Este repositório contém exercícios simples desenvolvidos durante o aprendizado de programação em **linguagem C**.

## Arquivos

### calculadora.c
Calculadora simples que permite realizar:
- Soma
- Subtração
- Divisão
- Multiplicação

### sistema_notas.c
Programa que calcula a média de um aluno e verifica:

- Aprovado
- Exame final
- Reprovado

### ordenacao_numeros.c
Programa que recebe **três números** e mostra:

- Maior número
- Número intermediário
- Menor número

### numero_por_extenso.c
Programa que recebe um número **entre 0 e 99** e mostra o número **por extenso**.

Exemplo:

Entrada:
25

Saída:
vinte e cinco

## Como compilar

Usando GCC:

gcc calculadora.c -o calculadora
gcc sistema_notas.c -o notas
gcc ordenacao_numeros.c -o ordenar
gcc numero_por_extenso.c -o extenso

## Como executar

./calculadora
./notas
./ordenar
./extenso

## Objetivo

Praticar conceitos básicos de programação:

- Entrada e saída de dados
- Estruturas condicionais
- Operações matemáticas
- Arrays
